\textbf{FAIR Genomes semantic metadata schema}
\newline

The FAIR Genomes semantic metadata schema to power reuse of NGS data in research and healthcare. Version 0.3-Minor, 2021-04-23. This model consists of 9 modules that contain 109 metadata elements and 84910 lookups in total (excluding null flavors).
